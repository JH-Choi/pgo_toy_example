# pgo_toy_example
![](./intro.gif)
- A toy example of Pose Graph Optimization (PGO) using g2o library.

## Pose Graph Information
- Pose: SE3Quat
- Vertex: VertexSE3Expmap
- Edge: EdgeSE3Expmap
- Information Matrix: Randomly generated

## Color Information
- Black: Current Estimated Poses
- Gray: Previous Poses
